# GoFrame项目结构

文档：[项目结构](https://goframe.org/start/index)

├── app  //业务逻辑层
│   ├── api //业务接口
│   ├── model //数据模型
│   └── service //逻辑封装
├── boot //初始化包
├── config //配置管理
├── docker //镜像文件
├── document //项目文档
├── i18n //I18N国际化	
├── library //公共库包
├── packed //打包目录
├── public //静态目录
├── router //路由注册
├── template //模板文件
├── vendor //第三方包
├── Dockerfile //镜像描述
├── go.mod //依赖管理
└── main.go //入口文件
